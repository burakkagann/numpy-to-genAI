1.1 Grayscale Color Basics
==========================

Introduction to digital color theory and pixel-level image representation.

This section introduces the fundamental concepts of digital color, starting with grayscale images and progressing to RGB color spaces. Understanding these basics is essential for all pixel-level manipulations in generative art.
